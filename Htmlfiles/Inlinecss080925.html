====Inline CSS====

Inline css is defined as the designs and styles given within the element tag. Example: check this content, its style is defined in the element tag"p" itself.

===Internal CSS===

Internal CSS is styles and designs defined within the head tag. Example: check this contant, its style given in the head tag

===External CSS===

External css is the styles and designs given in the other CSS file is linked to the html file. Example: Create a CSS file with style as required to be applied in the html file. create a html file in which you want this styles to be applied and then link the css file to this html file by using "Link" command in the html's head tag.
