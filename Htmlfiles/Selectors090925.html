Simple Selectors - Class, id, group, universal

Class Selector: styles defined within the inline tag by creating a 'class' with 'class name' and that particular class name is called by dot (.) in the css file that is linked. We can give more than one class name seperated by sapce within the same tag and also the same class name for different tags. Example: Refer button tag for clarification.

id selector

id selector is defined or called by (#) symbol followed by "id name" in css file.

Universal selector: is used to create designs or styles to the whole/entire web page and is called or defined by (*) symbol. No name is asigned to universal selector but we assign a name to Class and id selectors.
